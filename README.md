# Prompt Qwen Plugin

This is a plugin for Claude Code that provides enhanced prompt management capabilities.

## Features

- Meta-prompting tools for complex workflows
- Todo management for tracking tasks
- Context handoff capabilities
- Cross-platform compatibility

## Installation

```bash
./install.sh
```

## Usage

The plugin provides several slash commands for enhanced productivity:

- `/create-prompt` - Create new prompts with proper structure
- `/run-prompt` - Execute saved prompts
- `/add-to-todos` - Add tasks to your todo list
- `/check-todos` - Check your current todo list
- `/whats-next` - Create context handoff documents

## Configuration

The plugin is configured through TOML files in the meta-prompting and todo-management directories.

## Uninstall

```bash
./uninstall.sh
```
